Features

Everything Claude Code forgets, recalled.

Automatic capture with smart relevance scoring, privacy-first storage, and instant search across every session you've ever run.

🔍

Full-Fidelity Recovery

Resume any session with complete context — verbatim prompts, assistant responses, every tool call with inputs and outputs. Up to 1M token recovery budget.

🧠

Smart Relevance Scoring

Every tool use is scored 0.0–1.0 before storage. Write ops, errors, and architecture discussions score high. Config reads and empty results score low.

🔒

Auto-Redaction

8 secret patterns — API keys, bearer tokens, AWS keys, private keys, passwords, GitHub/OpenAI/Slack tokens — automatically scrubbed before storage.

📅

Date-Aware Search

Search with natural language dates: since="3 days ago", until="last friday", or ISO ranges. Cross-project search across all your repos.

💬

Inter-Session Messaging

Claude Code sessions can send messages to other projects. Messages route through the shared DB with operator approval — coordinated AI workflows.

⚡

Zero Overhead

No background daemons, no HTTP servers, no AI compression calls. Direct SQLite writes via bun:sqlite in WAL mode. Your token budget goes to actual work.
